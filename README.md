# Career Fit AI

中文实习/校招场景下的 AI 简历匹配助手。用户按维度填写求职画像，并粘贴目标岗位 JD，工具会按固定评分规则输出总匹配度、维度得分、投递建议、证据、缺口、简历改写建议和面试准备问题。

## 为什么做这个项目

很多求职工具只给出一个 ATS 分数，但中文实习和校招场景中，求职者真正需要的是判断：

- 这个岗位到底适不适合投；
- 我的背景里哪些经历能支撑 JD；
- 院校、专业、实习、项目成果、技能工具、岗位理解分别强弱在哪里；
- 哪些表达需要改写；
- 面试前应该准备哪些问题。

本项目定位为求职者工具，不是招聘方筛选系统。

## 当前功能

- 按维度填写中文求职画像
- 输入学校名称后自动识别 985/211/双一流标签
- 粘贴中文岗位 JD
- 输出 0-100 总匹配度
- 判断投递建议：强投 / 可投 / 谨慎投
- 按维度输出分数、证据、缺口和建议
- 给出简历改写建议
- 生成面试准备问题

当前评分维度：

| 维度 | 权重 |
| --- | ---: |
| 学历/院校背景 | 15 |
| 专业相关性 | 10 |
| 实习经历匹配 | 30 |
| 项目成果与量化证据 | 20 |
| 技能工具匹配 | 15 |
| 行业/岗位理解 | 10 |

默认使用本地规则分析，不产生 OpenAI API 费用。如果用户选择 `OpenAI 深度分析`，需要在页面侧边栏输入自己的 OpenAI API key。

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

可选配置默认模型：

```powershell
$env:OPENAI_MODEL="gpt-5-mini"
```

## 产品边界

当前版本不支持：

- PDF/DOCX 上传
- 多份简历批量比较
- 登录与历史记录
- 自动生成完整简历
- 保存或托管用户 API key

这些功能会在核心分析逻辑稳定后再考虑。
